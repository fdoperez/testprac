# Práctica 1 - Inferencia Condicional en Distribuciones Discretas Binarias
La práctica consiste en mplementar un programa que permita calcular **distribuciones condicionales** a
partir de una **distribución conjunta discreta** de variables binarias, utilizando
marginalización de variables.
El programa permitirá generar o cargar la distribución, seleccionar variables de
interés y condicionadas de forma intuitiva, calcular automáticamente la distribución
condicional y analizar el tiempo de ejecución en función del número de variables de
interés y de condicionadas. Para más información vease el [enunciado](./to-do/p1.pdf).

## Compilación y Ejecución

```bash
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ../bin/pr1_IAA
```
